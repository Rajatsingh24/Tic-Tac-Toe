# Tic Tac Toe

## Specification

* Website using HTML , CSS , JS

* Modes -> PvP and PvE 

* PvE is based on Minmax Theorem

* Responsive website

![](https://www.linkpicture.com/q/Screenshot-2023-03-11-194721.png)

## How to Play 

* Choose the mode : PvP(Player vs Player ) or PvE (Player vs Enemy )

* Click on the Board

* Enjoy the game
